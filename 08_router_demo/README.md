# router-demo
## 説明
このプログラムはアクター システムをセットアップし、ラウンド ロビン、ランダム、ConsistentHash、およびブロードキャスト プールという 4 つの異なるルーティング戦略を示します。各戦略は、アクターのプール間でメッセージを配布するために使用されます。カスタム ハッシュ関数を含む構造 `myMessage` 体は、メッセージ ルーティングの動作を示すために使用されます。
