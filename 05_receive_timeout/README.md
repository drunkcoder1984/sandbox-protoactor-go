# actor-receive-timeout
## 説明
このプログラムでは、SetReceiveTimeout を使用して、指定された時間内にメッセー ジが受信されなかった場合にアクションをトリガーするアクタを設定します。また、受信タイムアウトをリセットしないカスタム・メッセージ・タイプ NoInfluence を導入します。この例は、アクターベースのシステムにおけるタイムアウト処理を理解するのに役立ちます。
